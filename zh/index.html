Static PHP

在 Linux、macOS、FreeBSD、Windows 上与 PHP 项目一起构建独立的 PHP 二进制文件，并包含流行的扩展。

开始使用

静态 CLI 二进制

您可以轻松地编译一个独立的 PHP 二进制文件以供通用使用，包括 CLI、FPM SAPI。

Micro 自解压可执行文件

您可以编译一个自解压的可执行文件，并将 PHP 源代码与二进制文件打包在一起。

依赖管理

static-php-cli 附带依赖项管理，支持安装不同类型的 PHP 扩展。
